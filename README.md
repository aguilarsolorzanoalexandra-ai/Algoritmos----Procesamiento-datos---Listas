Algoritmos — Procesamiento de Datos — Listas

Repositorio que contiene 13 ejercicios prácticos desarrollados en Arduino (C++) enfocados en el manejo y procesamiento de listas.

Lista de Ejercicios

1. Cargar lista por consola  
2. Cargar lista con valores aleatorios  
3. Mostrar lista  
4. Obtener valor en posición M  
5. Modificar valor en posición M  
6. Intercambiar posiciones M y Q  
7. Buscar valor y retornar posición  
8. Sumar todos los valores  
9. Obtener valor máximo  
10. Calcular promedio  
11. Eliminar duplicados  
12. Invertir lista  
13. Ordenar lista ascendente
